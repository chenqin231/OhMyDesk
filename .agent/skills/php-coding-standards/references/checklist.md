## 代码提交前检查清单

### 必须检查项
- [ ] 所有文件开头有 `declare(strict_types=1);`
- [ ] 所有函数/方法有类型声明
- [ ] 所有用户输入已验证和过滤
- [ ] 所有 SQL 使用预处理语句
- [ ] 所有输出已正确转义
- [ ] 无硬编码敏感信息
- [ ] 异常有恰当处理
- [ ] 代码符合 PSR-12 规范

### 安全检查项
- [ ] 无 SQL 注入风险
- [ ] 无 XSS 风险
- [ ] 无 CSRF 风险
- [ ] 文件上传已验证
- [ ] 密码使用 `password_hash()`

### 性能检查项
- [ ] 无 N+1 查询问题
- [ ] 数据库查询已优化
- [ ] 大数据集使用分页或生成器

---

**维护说明**：
- 本 skill 适用于 PHP 8.0+ 项目
- 基于 PSR-12 编码规范
- 关注安全性和性能
